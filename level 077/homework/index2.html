ვოიდ და ჩვეულებრივი ფუნქციები

რა არის return-ი

იმისთვის, რომ გავიგოთ, რა არის ვოიდ და ჩვეულებრივი ფუქციები, თავდაპირველად უნდა ვიცოდეთ, რა არის return-ი და რაში ვიყენებთ მას. return არის სპეციალური სიტყვა(keyword), რომლითაც ჩვენ ვაბრუნებთ ფუნქციიდან მნიშვნელობას. მნიშვნელობის დაბრუნებაში იგულისხება, რომ როდესაც ჩვენ ფუქციას გამოვიძახებთ, მის ადგილას ჩაჯდება ის შედეგი, რასაც მოგცემს მოცემული ფუნქცია.

ჩვეულებრივი ფუნქცია

ჩვეულებრივი ფუნქცია უბრუნებს მნიშვნელობას პროგრამას, ანუ მას კოდის ბლოკში უწერია return სიტყვა. ასეთი ფუნქციის მიერ მოცემული შედეგი, ჩვეულებრივ შეგვიძლია გამოვიყენოთ პროგრამაში.

ვოიდ ფუნქცია

ვოიდ ფუნქცია არ აბრუნებს მნიშვნელობას, ის უბრალოდ პრინტავს მას. დაპრინტვის შემთხევაში, ფუნქციის შედეგი, ასე ვთქვათ ვარდება პროგრამის გარეთ(ტერმინალში), ხოლო თვითონ პროგრამაში არაფერი გვრჩება, ამიტომაც ჩვენ არ შეგვეძლება ფუნქციის მიერ მიღებული შედეგის გამოყენება.
